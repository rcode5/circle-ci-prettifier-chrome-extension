Update CircleCI status messages

This extension allows you to update the text used on CircleCI for failed and successful builds and for the word "branches".

Simply add your replacement values in the form below, click "Save" and go back to CircleCI Jobs or Workflows pages to see your prettier CircleCI dashboard.

This script is only active on pages that match 

  • https://circleci.com/gh/*
  • https://circleci.com/dashboard/*

So if you're not using Github projects it may not be as useful.
